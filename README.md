# 📘 Customer Churn Prediction System

## 🌟 Project Overview

This project is a machine learning based customer churn prediction system.

The model predicts whether a customer is likely to leave or stay using behavioral and service usage data.

The project is developed using:

- Streamlit
- :contentReference[oaicite:1]{index=1}
- Streamlit Cloud

---

##  Business Objective

Customer churn prediction helps companies:

- Improve customer retention
- Reduce marketing cost
- Increase revenue stability

This project demonstrates an end-to-end machine learning pipeline.
